# 📚 Library Management — Управление библиотекой

Модуль для управления библиотекой, созданный с целью изучения основ разработки в Odoo.

---

## 📋 Описание

Система управления библиотекой позволяет вести учет книг, авторов и читателей. Модуль демонстрирует базовые возможности разработки в Odoo: создание моделей, расширение существующих моделей, работу с ORM, вычисляемые поля и ограничения данных.

**Статус:** ✅ Завершён  
**Версия:** 1.0  
**Odoo:** 19.0  
**Лицензия:** LGPL-3

---

## ✨ Возможности

### 📖 Управление книгами
- Учет книг с полной информацией:
  - Название
  - Год издания
  - Количество страниц
  - Описание
- Привязка книг к авторам (Many2many связь)
- Автоматический расчет возраста книги (вычисляемое поле)
- Валидация данных:
  - Год издания не может быть в будущем
  - Количество страниц должно быть положительным числом

### ✍️ Управление авторами
- База данных авторов:
  - Имя
  - Дата рождения
  - Биография
- Автоматический подсчет количества книг автора
- Вычисление возраста автора
- Валидация вводимых значений

### 👤 Читатели библиотеки
- Расширение стандартной модели контактов (`res.partner`)
- Новые поля:
  - Чекбокс "Посетитель библиотеки"
  - Список взятых книг (One2many)
  - **Автоматический подсчет количества взятых книг**
- Ограничения:
  - Не более 5 книг на одного читателя
  - Валидация при сохранении

## 🗂️ Модели данных

### 1. `library.book` — Книги

| Поле | Тип | Описание |
|------|-----|----------|
| `name` | Char | Название книги |
| `publication_year` | Integer | Год издания |
| `pages` | Integer | Количество страниц |
| `description` | Text | Описание книги |
| `author_ids` | Many2many | Связь с авторами |
| `age_years` | Integer | Возраст книги (вычисляемое) |

---

### 2. `library.author` — Авторы

| Поле | Тип | Описание |
|------|-----|----------|
| `name` | Char | Имя автора (обязательное) |
| `birth_date` | Date | Дата рождения |
| `biography` | Text | Биография автора |
| `book_ids` | Many2many | Книги автора |
| `book_count` | Integer | Количество книг (вычисляемое) |

---

### 3. `res.partner` (расширение) — Контакты

**Добавленные поля:**

| Поле | Тип | Описание |
|------|-----|----------|
| `is_library_member` | Boolean | Отметка "Посетитель библиотеки" |
| `borrowed_book_ids` | Many2many | Взятые книги |
| `borrowed_books_count` | Integer | Количество взятых книг (вычисляемое) |

---

## 🖼️ Представления

### Form Views (Формы)
- **Книга:** Полная информация о ней
- **Автор:** Биография, дата рождения, список книг
- **Контакт:** Расширенная форма с вкладкой "Библиотека"

### List Views (Списки)
- **Книги:** Название, авторы, год издания, страницы
- **Авторы:** Имя, дата рождения, количество книг

### Search Views (Поиск)
- Фильтры по году издания
- Поиск по названию, автору
- Группировка по авторам

## 🚀 Установка

1. Скопируйте модуль в папку `addons/`
2. Перезапустите Odoo:
   ```bash
   docker-compose restart odoo
   ```
3. Обновите список приложений (Apps → Update Apps List)
4. Найдите "Управление библиотекой" и нажмите **Install**

## 🐛 Известные ограничения

- Нет системы ролей (все пользователи имеют одинаковые права)
- Нет истории выдачи книг (только текущее состояние)
- Нет интеграции с календарём для отслеживания сроков возврата
