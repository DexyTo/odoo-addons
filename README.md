# 🎓 Odoo Learning Projects

Учебные модули, созданные для изучения основ разработки в Odoo 19.

## 📋 О проекте

Этот репозиторий содержит два учебных модуля, демонстрирующих различные аспекты разработки в Odoo:
- Работу с моделями данных и ORM
- Систему прав доступа и безопасности
- Автоматизацию бизнес-процессов
- Улучшение пользовательского интерфейса

## 📦 Модули

### 1. [Library Management](addons/library_management/) — Управление библиотекой
**Статус:** ✅ Завершён  
**Сложность:** ⭐⭐☆☆☆

### 2. [Space Booking](addons/space_booking/) — Бронирование помещений
**Статус:** ✅ Завершён  
**Сложность:** ⭐⭐⭐⭐☆

Продвинутый модуль для управления резервированием различных типов помещений (переговорные, коворкинги, учебные классы, конференц-залы и др.) с акцентом на безопасность, автоматизацию и удобный интерфейс:

- **Гибкая система прав доступа** – пользователи работают только со своими бронированиями, менеджеры имеют полный контроль.
- **Автоматическая проверка пересечений** – предотвращает двойное бронирование одного помещения.
- **Валидация времени** – минимальная длительность 10 минут, максимум 24 часа, бронирование возможно не ранее чем за 10 минут до текущего момента.
- **Интеграция с календарём Odoo** – при подтверждении брони создаётся событие в календаре, автоматически синхронизируется при изменении времени, удаляется при завершении/отмене.
- **Напоминания** – поддержка стандартных напоминаний календаря (по умолчанию за 15 минут).
- **Cron-задача** – автоматическое завершение активных бронирований с истёкшим временем окончания (запуск каждые 10 минут).
- **Улучшенный UI** – стилизованные формы с градиентами, анимациями, цветовое кодирование статусов, календарный и графический просмотр.

**Ключевые функции:**
- Бронирование помещений с выбором типа и местоположения
- Полный жизненный цикл: черновик → активно → завершено / отменено
- Автоматическое создание и синхронизация событий календаря
- Аналитика: группировка по помещениям, статистика загрузки
- Богатые представления: список, форма, календарь, график, сводная таблица

[📖 Подробная документация →](addons/space_booking/README.md)

## 🚀 Быстрый старт

### Требования
- Docker и Docker Compose
- Git

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/DexyTo/odoo-addons.git
cd odoo-learning
```

2. **Запустите Odoo через Docker:**
```bash
docker-compose up -d
```

3. **Откройте браузер:**
```
http://localhost:8069
```

4. **Создайте базу данных:**
   - Email: `admin@example.com`
   - Password: `admin`
   - Database name: `odoo`

5. **Установите модули:**
   - Перейдите в **Apps** (Приложения)
   - Найдите и установите:
     - `Управление библиотекой`
     - `Бронирование помещений`

### Остановка

```bash
docker-compose down
```

### Полная очистка (удаление данных)

```bash
docker-compose down -v
```

## 🧪 Тестирование

Модуль **Meeting Room Booking** включает unit-тесты:

```bash
# Запуск всех тестов модуля
docker-compose run --rm web odoo \
    -d odoo \
    --db_host db \
    --db_user odoo \
    --db_password odoo \
    --test-enable \
    --stop-after-init \
    -u meeting_room_booking
```

**Результат:** 22 тестов покрывают модели, ограничения и права доступа.

## 📚 Что изучено

### Базовые концепции (Library Management)
- ✅ Создание моделей (`models.Model`)
