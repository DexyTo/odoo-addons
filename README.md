# 🎓 Odoo Learning Projects

Учебные модули, созданные для изучения основ разработки в Odoo 19.

## 📋 О проекте

Этот репозиторий содержит два учебных модуля, демонстрирующих различные аспекты разработки в Odoo:
- Работу с моделями данных и ORM
- Систему прав доступа и безопасности
- Автоматизацию бизнес-процессов
- Улучшение пользовательского интерфейса

## 🚀 Быстрый старт

### Требования
- Docker и Docker Compose
- Git

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/DexyTo/odoo-addons.git
cd odoo-learning
```

2. **Запустите Odoo через Docker:**
```bash
docker-compose up -d
```

3. **Откройте браузер:**
```
http://localhost:8069
```

4. **Создайте базу данных:**
   - Email: `admin@example.com`
   - Password: `admin`
   - Database name: `odoo`

5. **Установите модули:**
   - Перейдите в **Apps** (Приложения)
   - Найдите и установите:
     - `Управление библиотекой`
     - `Бронирование переговорных`

### Остановка

```bash
docker-compose down
```

### Полная очистка (удаление данных)

```bash
docker-compose down -v
```

## 🧪 Тестирование

Модуль **Meeting Room Booking** включает unit-тесты:

```bash
# Запуск всех тестов модуля
docker-compose run --rm web odoo \
    -d odoo \
    --db_host db \
    --db_user odoo \
    --db_password odoo \
    --test-enable \
    --stop-after-init \
    -u meeting_room_booking
```

**Результат:** 30 тестов покрывают модели, ограничения и права доступа.

## 📚 Что изучено

### Базовые концепции (Library Management)
- ✅ Создание моделей (`models.Model`)
- ✅ Типы полей (Char, Integer, Date, Text, Many2many)
- ✅ Вычисляемые поля (`@api.depends`)
- ✅ Ограничения (`_sql_constraints`, `@api.constrains`)
- ✅ Расширение моделей (`_inherit`)
- ✅ Базовые представления (Form, List, Search)

- ✅ Методы действий (`action_confirm`, `action_cancel`)
- ✅ Переходы между состояниями (Workflow)
- ✅ Cron задачи (автоматизация)
- ✅ Unit тестирование (`TransactionCase`)
- ✅ Улучшение UI (HTML/CSS)
- ✅ Стилизация форм и представлений
